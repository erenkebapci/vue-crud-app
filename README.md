# vue-crud-app
## How to Use

```shell
# Clone this repository
$ git clone https://github.com/erenkebapci/vue-crud-app

# Go into the repository
$ cd vue-crud-app

# Install json-server
$ npm i -g json-server

# Run database
$ json-server db.json

# Go into front-end with another terminal and install npm packages
$ npm i --s

# Start development server
$ npm run serve
